# Descripción
Minishell es un proyecto que consiste en la creación de nuestro propio shell en lenguaje C. 

El objetivo de realizar este proyecto es formarnos con el objetivo de entender:

* Uso de la librería *readline* de C
* Asignación y liberación de memoria
* Manejo de ficheros
* Creación de procesos y pipes
* Manejo de señales del sistema
* Comprobación de status
* Ejecución de comandos desde un programa
* Manejo de directorios, entorno y atributos
* Funciones como cd, env, pwd, unset, etc.
* Muchas otras cosas

La visión que nos aporta este proyecto es que mediante lenguaje C podemos interactuar con el sistema UNIX de una forma clara y sencilla. Dotándonos de los conocimientos y experiencias necesarias para el correcto manejo de las cosas anteriormente mencionadas para su realización en cualquier momento.
